Materi Lengkap tentang HTML

1. Pengertian HTML

HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML digunakan untuk memberikan struktur dan konten pada halaman web, termasuk teks, gambar, tautan, dan elemen multimedia lainnya.

2. Tujuan HTML

Tujuan utama HTML adalah:

  • Menyediakan struktur dasar untuk halaman web.
  • Mengorganisir konten dalam format yang dapat dibaca oleh browser.
  • Menentukan elemen-elemen seperti heading, paragraf, gambar, tautan, dan tabel.
  • Mendukung pengembangan web yang responsif dan interaktif.

3. Contoh Kode HTML

Berikut adalah contoh kode HTML sederhana yang menunjukkan berbagai elemen HTML.

Contoh 1: Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat Datang di Halaman HTML</h1>
    <p>Ini adalah paragraf pertama di halaman ini.</p>
</body>
</html>

Contoh 2: Menggunakan Elemen Tautan dan Gambar

<h2>Contoh Tautan dan Gambar</h2>
<a href="https://www.example.com">Kunjungi Example.com</a>
<br>
<img src="gambar.jpg" alt="Deskripsi Gambar" width="300">

Contoh Tautan dan Gambar

Contoh 3: Membuat Daftar

<h2>Daftar Belanja</h2>
<ul>
    <li>Beras</li>
    <li>Gula</li>
    <li>Minyak</li>
</ul>
