Materi Lengkap Belajar JavaScript

1. Variabel

Variabel digunakan untuk menyimpan data. Anda dapat menggunakan var, let, atau const untuk mendeklarasikan variabel.

let nama = "John"; // variabel dapat diubah
const umur = 30; // variabel konstan

2. Tipe Data

JavaScript memiliki beberapa tipe data, termasuk:

  • String: Teks, misalnya "Hello"
  • Number: Angka, misalnya 42
  • Boolean: Nilai true/false
  • Array: Daftar nilai, misalnya [1, 2, 3]
  • Object: Koleksi pasangan kunci/nilai, misalnya { nama: "John", umur: 30 }

3. Operator

Operator digunakan untuk melakukan operasi pada variabel dan nilai.

let a = 10;
let b = 5;
let hasil = a + b; // Penjumlahan

4. Fungsi

Fungsi adalah blok kode yang dapat dipanggil untuk melakukan tugas tertentu.

function salam(nama) {
    return "Halo, " + nama + "!";
}

5. Objek

Objek adalah tipe data yang menyimpan koleksi data dan entitas yang lebih kompleks.

let mobil = {
    merek: "Toyota",
    model: "Corolla",
    tahun: 2020,
    deskripsi: function() {
        return this.merek + " " + this.model + " (" + this.tahun + ")";
    }
};

6. Array

Array adalah struktur data yang digunakan untuk menyimpan beberapa nilai dalam satu variabel.

let angka = [1, 2, 3, 4, 5];
let angkaPertama = angka[0]; // Mengakses elemen pertama
